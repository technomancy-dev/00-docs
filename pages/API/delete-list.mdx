## Delete List (Pro Feature)

Delete a list via `delete` to the `/api/lists/:id` endpoint. Responds with `204`


### Path Parameters

| Param            | Type            | Required | Notes                                                                                                                                    |
|------------------|-----------------|----------|------------------------------------------------------------------------------------------------------------------------------------------|
| id               | uuid            | yes      | The lists uuid                                                                                                                  |

#### `id` (required)

**type:** `uuid`
