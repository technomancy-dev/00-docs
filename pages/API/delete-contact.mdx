## Delete Contact (Pro Feature)

Delete a contact via `delete` to the `/api/contacts/:id` endpoint. Responds with `204`


### Path Parameters

| Param            | Type            | Required | Notes                                                                                                                                    |
|------------------|-----------------|----------|------------------------------------------------------------------------------------------------------------------------------------------|
| id               | uuid            | yes      | The contacts uuid                                                                                                                  |

#### `id` (required)

**type:** `uuid`
