## Get Email (Pro Feature)

Get an email via `get` to the `/api/emails/:id` endpoint.

### Example
```json copy
{
  "data": {
    "id": "324492ec-f18a-4d41-97f5-d3a090641c0f",
    "from": "levi@double-zero.cloud",
    "subject": "hey",
    "to": [
      "bud2099@example.com"
    ],
    "cc": null,
    "bcc": null,
    "body": "<h1>Hi</h1>",
    "reply_to": null,
    "text": "Hi"
  }
}
```

### Path Parameters

| Param            | Type            | Required | Notes                                                                                                                                    |
|------------------|-----------------|----------|------------------------------------------------------------------------------------------------------------------------------------------|
| id               | uuid            | yes      | The emails uuid                                                                                                                  |

#### `id` (required)

**type:** `uuid`
